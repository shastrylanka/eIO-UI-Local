3. Cookies and web beacons

When Users visit the Sites, cookies and web beacons will be used by Yahoo as follows:

(a) Cookies

A cookie is a text-only string of information that a website transfers to the cookie file of User’s computer’s hard disk so that the website can recognise such computer when the User visit such web page later.

A cookie typically contains the name of the domain from which the cookie has come, the “lifetime” of the cookie, and a value, usually a randomly generated unique number.

When the User visits our Sites, Yahoo sends a cookie. In particular, Yahoo uses cookies for various purposes, namely to:

- recognize information related to the User’s Computer, so to provide customized content or remember the last visited page of a Yahoo product or service;

- keep track of preferences specified by the User while using Yahoo products or services;

- display the most appropriate advertising banners and content, based on the User’s interests and activity;

- provide general internal and customer analytics;

- conduct research to improve Yahoo's content, products and services;

- improve security measures such as requiring re-logging into a Yahoo product or service after a certain amount of time has elapsed;

- assist in identifying possible fraudulent activities.

User can configure his/her browser to accept all cookies, reject all cookies, or notify when a cookie is set. Each browser is different, so it is preferable that the User checks the "Help" menu of his/her browser to learn how to change his/her cookie preferences.

If all cookies are rejected, the use of Yahoo’s products or services by the User on the Sites that require a "sign in" may be jeopardized and, therefore, the User may not be able to take full advantage of all offerings available on Yahoo.

(b) Web beacons

Web pages may contain an electronic file, called “web beacon”, that allows a web site to count the number of users who have visited that page or to access certain cookies. Yahoo uses web beacons within its network in order to count the number of users and to recognise users’ computer, through cookies and only in case the User has already visited Yahoo. If the User rejects cookies, the web beacon tracking technology will still operate but it will not be able to link the activities of a specific browser from one session to another session.

(c) Other uses of Cookies and web beacons

Yahoo offers to the Advertisers tools that allow them to better understand the performance of their online marketing campaigns with Yahoo (“Yahoo! Analytic Tools”).

In order to use these Yahoo! Analytic Tools, an Advertiser may add Yahoo! Analytic Tools web beacons to its web sites. These web beacons are electronic files served by either perf.overture.com or yahoo.com that enable Yahoo to collect anonymous information from the users of the Advertiser's sites via cookies. These web beacons collect only anonymous non-personally identifiable information and the URL of the web page where the web beacon is located. Yahoo! maintains the data of each Advertiser separately.

The data collected are used to create aggregate reports for the Advertisers. No individual level data related to Users is shared with the Advertiser.

In addition to the above, Yahoo's email delivery vendor may use web beacons and cookies within HTML email messages sent on our behalf.

(d) Opt-out
It is possible to opt-out [Y1] from the collection of the information referred to in letter (c) above via web beacons. Cookies will need to be enabled in order to activate the opt-out function.

Note: This opt-out applies to a specific browser rather than a specific user. Therefore it will be necessary to opt-out separately from each computer or browser which is used. Additionally, the opt-out applies to web beacons only and not to other tools and cookies used by Yahoo.

5. IP addresses and information specifically related to your computer

When a User visits our Sites, Yahoo is allowed to log the corresponding IP address. In addition to this, Yahoo collects the information that the browser of the User’s computer automatically releases, such as information about the operating system and the browser version. When the browser or the email application used by the User requests a web page or email from another computer on the Internet, it automatically provides that computer with the address where it should send the information. This is called "IP address" (IP stands for "Internet protocol.") For most of the users accessing the Internet from a dial-up Internet service provider (ISP), the IP address will be different every time they log on. It is, however, possible to trace the user with the assistance of the ISP.

Yahoo, therefore, receives IP addresses from all Users because this information is automatically reported by the browser each time a User views a web page.

IP addresses may be used for various purposes, namely to:

· provide targeted advertisements and personalised services to the Users based on their IP address;

  • trace out fraudulent activity;

· identify service or technology problems reported by the Users or engineers that are associated with the IP addresses controlled by a specific web company or ISP;

· provide analytics with respect to the Users from specific countries or regions of the world;

  • log a request on our servers when a Yahoo web page is requested and viewed.
